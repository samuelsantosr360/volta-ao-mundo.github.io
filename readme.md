

# 🌏 Projeto Austrália - Volta ao Mundo

Este é um projeto web focado em apresentar informações culturais, geográficas e gastronômicas da **Austrália**. O site foi desenvolvido com um design moderno, responsivo e repleto de animações interativas para proporcionar uma excelente experiência ao usuário.

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as seguintes ferramentas:

* **HTML5** - Estrutura semântica do site.
* **CSS3** - Estilização personalizada com gradientes e layouts modernos.
* **Bootstrap 5** - Framework para garantir total responsividade em dispositivos móveis e organização de grid.
* **Animate.css** - Biblioteca para animações de entrada rápidas e fluidas.
* **AOS (Animate On Scroll)** - Animações disparadas conforme o usuário rola a página.
* **Google Fonts** - Tipografia focada na fonte *Poppins*.

## 📋 Funcionalidades

* **Navbar Responsiva:** Menu que se adapta para dispositivos móveis com efeito de transparência.
* **Botão Flutuante Interativo:** Um botão lateral que abre um Modal com informações do desenvolvedor.
* **Seção de Sabores (Cuisine):** Títulos com preenchimento de imagem (background-clip: text) e cards de pratos típicos.
* **Seção de Informações:** Grid responsiva exibindo dados gerais como capital, população e moeda.
* **Cards de Conteúdo:** Layout em colunas para facilitar a leitura sobre fauna, flora e economia.

## 📂 Estrutura de Pastas

```text
/projeto-australia
│
├── /Pages          # Contém as páginas HTML (sobre, sabores, cultura, etc)
├── /Style          # Arquivos de estilização CSS
├── /Imagens        # Imagens
└── index.html      # Página principal do projeto
```

## 🛠️ Como Executar o Projeto

1. Faça o download ou clone este repositório.
2. Certifique-se de estar conectado à internet (para carregar os links do Bootstrap e Google Fonts via CDN).
3. Abra o arquivo `index.html` em qualquer navegador moderno.
